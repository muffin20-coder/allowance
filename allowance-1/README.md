# Allowance CLI

Allowance is a command-line interface (CLI) application designed to help users manage their allowances and transactions effectively. This project provides a simple and intuitive way to track financial activities related to allowances.

## Features

- **Command-Line Interface**: Easily interact with the application through a user-friendly CLI.
- **User Interface**: A graphical user interface (GUI) for a more visual approach to managing allowances.
- **Balance Tracking**: View current balance and transaction history.
- **Transaction Management**: Add, edit, and delete transactions with ease.

## Installation

To install the Allowance CLI, clone the repository and install the required dependencies:

```bash
git clone <repository-url>
cd allowance
pip install -e .
```

## Usage

To start the CLI application, run the following command:

```bash
allowance
```

Follow the prompts to manage your allowances and transactions.
